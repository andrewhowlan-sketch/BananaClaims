# Banana Claims Roadmap

## v0.0.1 - Initial Command
- /claim command confirms the mod is loaded.

## v0.0.2 - Create Claims
- /claim create <name>
- Claims the chunk the player is standing in.

## v0.0.3 - Claim Storage
- Save claims to config/bananaclaims/claims.json
- Load claims on server start.

## v0.0.4 - Claim Info
- /claim info
- Shows claim name, owner, dimension, chunk, and description.

## v0.0.5 - Claim List
- /claim list
- Shows claims owned by the player.

## v0.1.0 - Core Claim System
- Create claims
- Delete claims
- Rename claims
- Descriptions
- Basic ownership
- JSON storage
